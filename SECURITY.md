# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in NEXUS, please report it responsibly:

1. **Do NOT open a public issue**
2. Email: security@nexus-project.dev
3. Include: description, reproduction steps, impact assessment

## Response SLA

- Acknowledgment: within 48 hours
- Initial assessment: within 7 days
- Fix timeline: based on severity

## Scope

- Anomaly gate bypasses
- Tenant isolation failures
- Seal/ledger tampering
- Authentication/authorization flaws
- Prompt injection that bypasses persona constraints
